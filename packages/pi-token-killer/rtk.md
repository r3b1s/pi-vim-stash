# RTK

Bash commands auto-rewritten through rtk for compressed output. No manual prefix needed.

```
rtk gain           # token savings
rtk discover       # missed optimization opportunities
rtk session        # RTK adoption in recent sessions
rtk proxy <cmd>    # raw command (no rewrite)
rtk rewrite <cmd>  # preview rewrite
```

Commands already starting with `rtk`, meta commands, interactive, heredocs: left alone.
